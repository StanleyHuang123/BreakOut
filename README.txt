Type make run in command line to start the game 
Press <- or Click left part of game screen to move paddle left
Press -> or Click right part of game screen to move padle right
The value of fps is shown at the left bottom of the game screen
The value of score is shown at the right bottom of the game screen

You can change size of game screen before the game at the code line 747, you can resize the window during the game
